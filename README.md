## 🎯 The Mission
**PacPay** isn't just a fintech app—it's a high-fidelity **Financial RPG**. We’ve merged the precision of institutional finance with the addictive mechanics of a 1980s arcade. Track your wealth, battle your spending ghosts, and climb the global leaderboards to become a **Master Saver**.

---

## 🕹️ System Modules

### 🏙️ The Command Center (Dashboard)
A premium **Bento Grid** HUD that provides a 360-degree view of your financial matrix.
- **Dynamic P1 HUD**: Real-time tracking of XP, Levels, and Rank.
- **Vault Capacitor**: Interactive spending threshold guards.
- **Recent Sessions**: Atomic logs of every credit and debit.

### 🧠 The Neural Link (AI Coach)
A resilient, 24/7 **AI Agent Manager** that serves as your financial guardian.
- **Dual-Core Resilience**: Seamlessly switches to **Local Virtual Intelligence (VI)** if the cloud uplink is interrupted.
- **Context-Aware Analytics**: Real-time advice based on your Rank, XP, and Behavior Score.
- **Uptime Guaranteed**: 100% availability for strategic coaching.

### 🍒 The Treasury (Rewards Hall)
Trade your **P1 Credits** for real-world loot in a neon-lit arcade vault.
- **Ghost Sentinels**: Locked rewards guarded by **Blinky, Pinky, Inky, and Clyde**.
- **Instant Redemption**: Real-time Firestore synchronization for reward claims.
- **Visual PWA HUD**: Glowing credit monitors and animated success notifications.

### 🏟️ The Trading Arena
A high-fidelity market simulation integrated directly into your dashboard.
- **Cyberpunk Feed**: A cinematic CSS-animated live market preview.
- **Portfolio Tracking**: Real-time asset valuation and growth charts.

## ⚙️ Game Mechanics

| Component | Logic | Impact |
| :--- | :--- | :--- |
| **XP Points** | Earned through manual logging & savings streaks. | Increases your Arcade Rank. |
| **Behavior Score** | Base 100. Drops when spending limits are breached. | Unlocks legendary Treasury vaults. |
| **Vault Capacitor** | Customizable weekly spending threshold. | Exceeding triggers the "Overspent" alert. |
| **Ghost Guards** | Level-based protection on rewards. | Disappears when you reach the required Rank. |

## 🛠️ Technical HUD

*   **Framework**: Next.js 16.2.4 (App Router Architecture)
*   **Aesthetics**: Vanilla CSS + Tailwind Core + Framer Motion
*   **Database**: Google Firebase Firestore (Real-time Sync)
*   **Intelligence**: Google Gemini 1.5 Flash + Local Fallback Engine
*   **Charts**: Chart.js (High-Fidelity Financial Visualization)
*   **PWA**: Workbox-powered offline support and mobile installation.

## 🚀 Uplink Instructions (Installation)

1. **Clone the matrix**:
   ```bash
   git clone https://github.com/jawaanquizziz/ITM_Hackathon_2026_4Hackers.git
   ```
2. **Fuel the engine**:
   ```bash
   npm install
   ```
3. **Configure Environment Tokens** (`.env.local`):
   ```env
   NEXT_PUBLIC_FIREBASE_CONFIG={...}
   GEMINI_API_KEY=your_key_here
   NEXT_PUBLIC_RAZORPAY_KEY_ID=your_key_here
   ```
4. **Boot the HUD**:
   ```bash
   npm run dev
   ```
